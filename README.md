# **Setup**

**Install**
- `npm install`

**Local development**
- `npm run start`

**Create Lambda**
- `npm run build-lambda`
- Upload created .zip file to Amazon Lambda Function
- Increase Lambda memory to min 1024m 
- Increase Lambda timeout to 5 min
